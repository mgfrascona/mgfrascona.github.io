# My JavaScript website!

One of my original website builds that I made in my JavaScript class!

# Features

## :boy: Profile

Learn about my goal, favorite movies, and favorite books in my profile section. Hopefully your favorites appear there too!

## :carousel_horse: Carousel

An automatic, working image carousel showcasing different pictures.

## :clock2: Time-of-Day Tracker

A greeting related to the time of day will appear depending on said time thanks to JavaScript code!

## :memo: To-Do List

Add your own items to a to-do list!

## :zap: Pokemon API

Who's that Pokemon?! Find out with this generator that randomly chooses a Pokemon from PokeAPI.
