# Michael Frascona — Web Design Portfolio

A personal web portfolio showcasing my work as a web designer and creative developer. Built from scratch in VS Code with a focus on artistic flair, responsive design, and interactive experiences.

🌐 **[View Live Site](https://mgfrascona.github.io/)**

---

## About

I'm a web designer and developer with a passion for building visually striking, user-friendly websites for small businesses and charity organizations. This portfolio demonstrates my range across corporate, tourism, hospitality, and nonprofit design.

---

## Features

- **Interactive 3D hero logo** — Custom gear and infinity symbol animation built in Blender, rendered in the browser via Three.js with UnrealBloomPass glow and RoomEnvironment metallic reflections. Click and drag to explore it.
- **Responsive layout** — Mobile-first design with a hamburger menu on smaller screens.
- **Skills section** — Organized by Design & Creativity, Frontend Development, Data Management, and Tools & Workflow.
- **Projects showcase** — Each project has its own page with a detailed case study, screenshots, and links to live demos and GitHub repos.
- **Contact form** — Functional email submission powered by Formspree.
- **Resume download** — One-click PDF download.

## Tech Stack

**Frontend**
- HTML5, CSS3, JavaScript (ES6+)
- Three.js (CDN importmap) — 3D rendering, GLB model loading, bloom post-processing
- Google Fonts

**3D & Animation**
- Blender — custom GLB logo animation
- Three.js — OrbitControls, GLTFLoader, EffectComposer, UnrealBloomPass, RoomEnvironment

**Tools**
- VS Code + Live Server
- Git & GitHub
- Figma / Adobe XD
- Formspree (contact form)

## Getting Started

No build tools or dependencies required. Just clone the repo and open with Live Server in VS Code.

```bash
git clone https://github.com/mgfrascona/mgfrascona.github.io.git
```

Then right-click `index.html` in VS Code and select **Open with Live Server**.
